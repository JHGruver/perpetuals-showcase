# Perpetuals Interview Showcase - Project Overview

## Purpose
This project demonstrates advanced frontend development skills and strategic thinking for the Chief Product Officer position at Perpetuals.com (NASDAQ: PDC).

## Project Structure

```
Perpetuals/
├── src/
│   ├── scss/                    # SASS source files
│   │   ├── utils/               # Variables, mixins, functions
│   │   ├── vendors/             # Third-party styles, resets
│   │   ├── base/                # Typography, global styles
│   │   ├── components/          # Reusable UI components
│   │   ├── layouts/             # Page layout structures
│   │   └── pages/               # Page-specific styles
│   ├── js/
│   │   ├── animations/          # GSAP, Three.js, particles
│   │   ├── utils/               # Helper functions
│   │   └── components/          # JS component logic
│   └── assets/
│       ├── images/              # All image assets
│       │   ├── logos/           # Brand logos
│       │   ├── icons/           # UI icons
│       │   ├── backgrounds/     # Background images
│       │   └── ui/              # UI elements
│       ├── fonts/               # Self-hosted fonts
│       └── videos/              # Video assets
├── sites/
│   ├── homepage-rebuild/        # Redesigned homepage
│   ├── features-rebuild/        # Redesigned features page
│   ├── competitive-analysis/    # Market analysis site
│   ├── design-system/           # Component documentation
│   └── bonus-site/              # Additional showcase
├── docs/                        # Documentation
├── reports/                     # Research reports
├── CHANGELOG.md                 # Version history
├── package.json                 # Dependencies
└── vite.config.js              # Build configuration
```

## Sites Included

### 1. Homepage Rebuild
A complete redesign of perpetuals.com homepage with:
- Immersive particle animations
- Parallax scrolling effects
- Digital storytelling narrative
- Enhanced visual hierarchy
- Preserved brand identity

### 2. Features Page Rebuild
Kronos X platform features showcase with:
- Interactive feature exploration
- 3D product visualizations

- Micro-interactions

### 3. Competitive Landscape Analysis
Interactive analysis of trading platform design:
- Robinhood, TD Ameritrade, Schwab, etc.
- Design pattern comparison
- UX friction analysis
- Accessibility review

### 4. Design System Website
Atomic design documentation:
- Atoms (colors, typography, icons)
- Molecules (buttons, inputs, cards)
- Organisms (headers, forms, sections)
- Templates and pages

## Technology Stack

### Build Tools
- **Vite** - Fast build tool with HMR
- **SASS** - CSS preprocessing

### Animation Libraries
- **GSAP** - Professional-grade animation
- **Three.js** - 3D graphics and WebGL
- **Particles.js** - Particle effects
- **Locomotive Scroll** - Smooth scrolling
- **Lenis** - Scroll smoothing
- **Splitting.js** - Text animations
- **Swiper** - Touch-enabled sliders

### Design Principles
- Mobile-first responsive design
- Fluid typography and spacing
- Accessibility (WCAG 2.1 AA)
- Performance optimization
- Progressive enhancement

## Running the Project

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
